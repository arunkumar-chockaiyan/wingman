# Wingman

Wingman is a real-time sales assistant application designed to provide intelligent orchestration using Kafka and Prisma.

## Features
- Real-time event orchestration via Kafka.
- Seamless data management with Prisma and PostgreSQL.
- Agentic capabilities for sales assistance.

## Getting Started
1. Clone the repository.
2. Install dependencies: `npm install`.
3. Set up your environment variables.
4. Run the application: `npm run dev`.
